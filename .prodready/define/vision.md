# Vision

## Problem Statement
Personal trainers lack a dedicated tool to create structured training plans for clients, schedule sessions on a calendar, and capture real-time progress during live workouts. Generic tools (spreadsheets, notes apps) are not purpose-built for this workflow.

## Target Users
- **Trainers** (primary): Personal trainers managing ~20–25 clients each, who need to plan, schedule, and run training sessions
- **Clients** (secondary): End-users who log in to review their past workouts

Multi-tenant platform — many trainers can sign up independently.

## Core Value Proposition
Fancy, intuitive workout scheduling combined with a live session mode that lets trainers capture unexpected progress (e.g., client lifted more weight than planned) in real time.

## Success Metrics
- Number of active paying subscriptions (post-billing launch)
- Weekly active trainers (sessions logged, plans created)

## MVP Scope

### Must Have (MVP)
- Trainer sign-up and authentication
- Client management (add / invite clients)
- Calendar-based workout scheduling (click day + hour, define exercises)
- Exercise builder (sets, reps, weight per exercise)
- Live training session mode (start session, log real-time progress notes / unexpected weight increases)
- Client portal (view past workouts only)
- Dark mode, premium UI with custom typography

### Nice to Have (Future)
- Client feedback / side-effect reporting after sessions
- Progress charts (weight-over-time per exercise)
- Subscription and billing system
